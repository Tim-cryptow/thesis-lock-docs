ThesisLock's authoritative history lives in the product repository. This page collects the canonical links and points you at the upstream changelog.

## Changelog

The product app publishes its own changelog at [thesis-lock.vercel.app/docs/changelog](https://thesis-lock.vercel.app/docs/changelog), and release history is tracked in the [product repository](https://github.com/Tim-cryptow/thesis-lock). Treat those as the source of truth for product changes. This documentation site tracks the protocol and its interfaces; when contract signatures, SDK methods, CLI flags, or API routes change upstream, the corresponding reference pages here are updated to match.
