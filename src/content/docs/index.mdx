## What ThesisLock does

ThesisLock is a hash-anchoring proof-of-existence service. You drop a file into the web app; your browser hashes it with SHA-256 entirely on your device; you sign a Stacks transaction that records the hash on chain with an optional label. The document itself never leaves your machine. Anyone can later visit `/v/<hash>` to confirm when the anchor was created, which wallet created it, and what label was attached, then re-upload a file to check that its hash matches.

Stacks settles to Bitcoin, so each anchor inherits Bitcoin-grade finality. The chain is always the source of truth.
